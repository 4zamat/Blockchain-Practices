Task 10
Write a contract that contains one function.
This function takes one argument - a list of integers.
The function returns the sum of all elements in the list.

Task 11
Write a contract that contains one function.
This function takes one argument - a string.
The function returns the same string but reversed.

Task 12
Write a contract that contains one function.
This function takes one integer argument - n.
The function returns true if n is a perfect square (e.g., 4, 9, 16) and false otherwise.


---

Task 13
Write a contract that contains one function.
This function takes one argument - an integer n.
The function returns true if n is a palindrome (it reads the same backward as forward), otherwise returns false.


---

Task 14
Write a contract that contains one function.
This function takes an array of integers as an argument.
The function returns the maximum value in the array.

Task 15
Write a contract that contains one function.
This function takes one argument - a string.

Task 16
Write a contract that contains one function.
This function takes one argument - an array of integers.

Task 17
Write a contract that contains one function.

Task 18
Write a contract that contains one function.
This function takes two integer arguments, base and exponent.

Task 19
Write a contract that contains one function.

Task 20
Write a contract that contains one function.
This function takes an integer n as an argument.
The function returns a list of all prime numbers less than or equal to n.
